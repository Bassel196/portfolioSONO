About the Project

Sonocardia is an intelligent heart sound analysis system that uses deep learning to detect heart murmurs and predict detailed murmur characteristics. Our system provides accurate patient-level analysis based on audio recordings from multiple auscultation points, helping to improve early heart disease detection. The system features real-time murmur detection, classification of murmur types, and integrates with mobile applications and embedded hardware for remote monitoring.

Why Sonocardia?

Cardiovascular diseases claim 17.9 million lives annually — that’s 32% of all global deaths.

Sonocardia is revolutionizing cardiac care through real-time, AI-driven heart monitoring, making early detection accessible, affordable, and automatic.

What is Sonocardia?

Sonocardia is an AI-powered mobile application that enables real-time heart sound and ECG analysis to detect and predict heart diseases quickly and accurately, using an embedded hardware.
  • Captures heart sounds using a modified stethoscope (MAX9814 + ESP32)
  • Records ECG signals via AD8232 sensor
  • Analyzes both in real-time using AI and deep learning
  • Detects murmurs, arrhythmias, heart failure risk, and more
  • Notifies users and emergency services in critical cases
  • Enables users to share results directly with doctors

How It Works

  • Record: 5–10 seconds of heart sound and ECG via the app
  • Analyze: ML & DL models trained on 100k+ real-world samples
  • Diagnose: Classify as normal/abnormal, predict severity, alert user
  • Share: Instantly send reports to physicians or caregivers

Technology Highlights

  • Machine & Deep Learning (CNNs, Random Forest, Transformers)
  • Accuracy: +90% in HeartSound Classification and in early Arrhythmia Detection and Prediction
  • Integrated hardware for real-time recording & feedback
  • Seamless cloud communication and data visualization
  • Built-in security & emergency response capabilities

Target Users
